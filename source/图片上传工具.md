## Intro
基于node图片上传工具，将图片上传到七牛仓库，获取在线图片地址。 主要是为了在写mackdown时，插入图片
[github地址](https://github.com/lsl233/qiniu)
<!-- more -->
## Installation
```
git clone 'https://github.com/lsl233/qiniu.git'
cd qiniu
npm install
npm run start
```

## Usage
```
path:
```
输入图片路径，mac直接拖入
```
name:
```
选填，重新设置图片名字， 默认为图片名字。
